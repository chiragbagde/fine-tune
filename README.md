# Fine-tuning LLMs with LoRA

Experiments fine-tuning small language models on code generation tasks using LoRA and QLoRA.

Models tuned: TinyLlama 1.1B, Mistral 7B, Phi-2. Dataset: Python code instructions (18k Alpaca-format samples).
